### 📦 SideStore (Alternative – No PC after setup) 
**Install & refresh iPA directly on your device**

📝 **About SideStore:**
* Works like AltStore but refreshes apps over WiFi
* No need to reconnect to a PC every 7 days
* Uses an Apple ID (recommended to use a secondary account)
* Supports most iOS versions

📥 **Download & Guide:**
* [YouTube Tutorial](https://www.youtube.com/watch?v=kaYBV9_uvaA)
* [Official Website](https://sidestore.io)

⚙️ **Requirements:**
* iPhone / iPad
* Apple ID
* Computer (first time installation only)
* Connected to the same WiFi network

🚀 **Steps:**
1. Install SideStore using AltServer
2. Trust the app in your device Settings
3. Enable the VPN inside the SideStore application
4. Install & refresh apps directly on your device

📦 **Compatible iPA:**
* uYouPlus
* YTMUltimate
* Spotilife
* BHTikTok
* And more…

🔐 **Tips:**
* Keep SideStore refreshed
* Turn on the VPN when refreshing apps
* Use a secondary Apple ID for safety
* Reinstall if the application crashes

🚩 **Note:**
SideStore does not feature permanent signing like TrollStore; applications must be manually refreshed every 7 days.
